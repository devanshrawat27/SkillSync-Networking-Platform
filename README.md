## ✦ What is SkillSync?

SkillSync is a **full-stack student networking and collaboration platform** — replacing random group formation with skill-based, data-driven team building. Find the right teammates, build your team, and collaborate — all in one place.

---

## ⚡ Features

### 👤 Profile System
- Professional student profiles with skills, interests, and social/coding links
- Profile photo upload with visibility controls
- Mentor mode with experience, bio, and expertise tags

### 🤝 Networking & Team Formation
- Discover peers by skill and domain relevance
- Send, accept, and manage connection requests
- Browse public projects and request to join teams

### 💬 Collaboration Tools
- Real-time messaging between connected users
- In-app notifications for requests and updates
- Project feeds to keep teams aligned

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- A Supabase project ([supabase.com](https://supabase.com))

### Installation

```bash
# Clone the repository
git clone https://github.com/devanshrawat27/SkillSync-Networking-Platform.git
cd SkillSync-Networking-Platform

# Install dependencies
npm install
```

### Environment Setup

Create a `.env` file in the project root:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_publishable_key
```

### Run

```bash
npm run dev
```

App runs at `http://localhost:8080`
